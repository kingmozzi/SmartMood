### 포트 연결 후

포트 연결 후 아래의 버튼들이 활성화 됨.

Live Mode - Display와 Sound에 실시간으로 반응함.

Mood Mode - 아래의 RGB에 원하는 값을 입력후 Submit을 눌러 원하는 무드 조성.

진동 on/off - 진동효과를 on off하는 토글 버튼

포트 해제 - LED를 끄고 연결된 포트와 연결을 해제함
